# Tauri Icon Assets

This directory contains the Tauri bundle icon set for Agentic Workbench.

`icon.png` is the checked-in source image for the generated platform icons. It is
a 512x512 transparent PNG. The files referenced by `src-tauri/tauri.conf.json`
are generated derivatives:

- `32x32.png`
- `128x128.png`
- `128x128@2x.png`
- `icon.icns`
- `icon.ico`

Additional size-specific files in this directory are also generated by the
Tauri icon command for platform packaging targets. Do not hand edit generated
icon derivatives. If the product icon changes, replace `icon.png` and regenerate
the full set in one commit:

```sh
pnpm --filter @yoophi/agentic-workbench tauri icon apps/agentic-workbench/src-tauri/icons/icon.png
```

The command writes output to the `icons` directory next to
`src-tauri/tauri.conf.json`.
